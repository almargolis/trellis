# Welcome to the Essays Garden

This is a sample essay to show how the multi-level garden structure works.

## Features

- Each directory under `content/` is a separate garden
- Each garden has its own `config.yaml` with title and description
- Gardens appear in the navigation menu
- Articles are organized within their respective gardens

## Getting Started

Create new gardens by adding directories under `content/` and include a `config.yaml` file.
